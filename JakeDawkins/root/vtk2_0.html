VTK (Video Toolkit)

Current Version: 2.0
VTK is a toolkit for video professionals, whether they be in live, filming, or editing. This tool allows the user to calculate timecode (potentially a very time consuming task otherwise), bitrate, and estimated file size for a number of formats.

NEW: A custom settings tab allows users to store their favorite encoding settings using an internal database and sqlite3.
NOTE: Due to time constraints, the validation of fields in the new and edit setting windows have not been implemented.
